projects/:id/graph/sprints
	GET trae todos los nodos hijos de un proyecto, hasta el nivel de sprints (project->release->sprints, es decir una union de project->release, project->sprint, release->sprint)
		todo lo que tiene un proyecto hasta los sprints, no traer, el team del sprint ni sus miembros, no traer baklog items y tareas

		consulta de apoyo: 
			MATCH (proj:Project {name:'Proyecto 1'})-[r]-(sprint:Sprint)
			RETURN proj as startNode, r as link, sprint as endNode
			UNION ALL
			MATCH (p:Project {name:'Proyecto 1'})-[r]-(r:Release)
			RETURN p as startNode, r as link, r as endNode
			UNION ALL
			MATCH (p:Project {name:'Proyecto 1'})--(release:Release)-[r]-(sprint:Sprint)
			RETURN release as startNode, r as link, sprint as endNode		campos, firma:
